# 📊 AI Data Analysis Dashboard

## 🚀 Project Overview

This project is an AI-powered data analysis dashboard built with Streamlit.

Users can:

* Upload CSV files
* Perform automatic data analysis
* Visualize data with charts
* Get AI-generated insights

---

## 🧠 Features

* Data preview
* Statistical summary
* Interactive visualization
* AI-powered interpretation (GPT API)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib / Seaborn
* OpenAI API

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
